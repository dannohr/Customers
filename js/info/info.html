Methods:

Update

To modify a record you need to send a request marked with __method[]=PUT and specify the id and the lock_version of that record plus a set of field values.

Create

To create a record you need to send a request marked with __method[]=POST and specify id=0 and lock_version=0 plus a set of fields to put in the record.

Delete

To delete a record you need to send a request marked with __method[]=DELETE and specify the id and the lock_version of a record to be deleted.
